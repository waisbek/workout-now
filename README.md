### WORKOUT NOW

I started to practice exercise at home during the COVID-19 quarantine period.

I saw an opportunity to enhancements my developer skills building this project.
A web application that allow to create a exercise plan, after that you can start a timer that will run according each activity in the plan.

######  This project was developed using:
- React as interface view library.
- Bootstrap and reactstrap for design/layout of the components
- react-router-dom as navigation library.
- uifx library to play sounds.
- react-circular-progressbar component to improve the UI (countdown timer).
